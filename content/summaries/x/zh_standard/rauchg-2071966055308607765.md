## Summary

Vercel 推出 **Vercel Services**，允许在单个项目中**混排**多种运行时：例如 Python 后端 API、ExpressJS 服务器和 React SPA 可共存于同一 Vercel 项目。

核心要点：
- 本地可用 `vc dev` 统一运行
- 部署与回滚**一次性完成**
- 统一观测、监控与调试
- 支持内部网络通信

这意味着开发者无需再为不同语言/框架拆分项目，可在一个 Vercel 项目内完成全栈服务的开发、部署与运维，简化多服务架构的管理复杂度。
