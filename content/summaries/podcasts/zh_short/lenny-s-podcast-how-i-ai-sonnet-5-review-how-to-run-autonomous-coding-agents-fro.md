**核心要点**
*   **基准必须可重复**：Claire 使用冻结输入、固定评分标准和相同任务构建了“How I AI Bench”，解决了以往无法跨时间比较模型的问题。

*   **按任务选模型，而非一刀切**：Claire 的加权指数显示，GPT-5.5 适合写 PRD，Sonnet 4.6 适合原型和闲聊，Opus 4.8 或 Sonnet 5 适合代码库导航。

*   **AI 压缩了信息优势**：Alessio 用 Codex 浏览器访问能力寻找被低估的宝可梦卡片，证明 AI 能将过去依赖人力和时间的规模化信息优势，压缩为一段精良的提示词。

**值得展开的细节**

*   Claude Code 能读取历史会话记录，为个人实际工作场景量身定制评估任务。
